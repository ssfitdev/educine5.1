Test Entities API

Testing: /api/entities (proxied to https://portal.ssfkerala.org/api/entities)

This will test the entities API with parentId query parameter for cascading data (districts → divisions → sectors).

API Format: GET /api/entities?parentId=xxx

⚠️ Authentication Required:
The entities API requires authentication. You need to set the ENTITIES_API_TOKEN environment variable in your Vercel deployment settings.

Steps:
  1. Get the API token from your backend team
  2. Go to Vercel Dashboard → Your Project → Settings → Environment Variables
  3. Add: ENTITIES_API_TOKEN = your-token-here
  4. Redeploy the site
